### 소속된 학과를 소개하는 앱

> - 여러 개의 액티비티를 만들고 각 페이지들 간의 이동을 어떻게 할 것인지에 초점
> - 홈으로 돌아가기는 옵션 메뉴를 이용하여 소개앱의 맨 첫페이지로 돌아가기를 구현하는 방식

`메인화면`

- 학과 소개
- 이미지 버튼 3개 : 교수진 소개, 학과 community소개, 공지사항

`2번째 activity`

- 교수님들에 대한 소개 페이지

`3번째 activity`

- 임의의 학생 커뮤니티를 소개하는 페이지

`4번째 activity`

- 본인의 이름을 포함한 소개 페이지

`5번째 activity`

- 학과 홈페이지에 가장 최근에 올라온 공지사항
